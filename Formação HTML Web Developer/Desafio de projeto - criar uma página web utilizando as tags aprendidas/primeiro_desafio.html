O que são Clients

O que são?

É o lado que consome os conteúdos disponíveis na internet. São responsáveis por iniciar a comunição e exibir ao usuários o conteúdo recebido do servidor.
Ex: navegadores, aplicações que usem API's e aplicativos mobiles.

O que são Servers?

O que são?

São computadores com alto poder computacional que fornecem os conteúdos solicitados pelos clientes, sendo ele páginas html, arquivos xml, arquivos json, entre outros.

Ex: Servidor FTP, servidor NTP, servidor de aplicação.

Como é uma estrutura básica de um HTML?

HTML é uma liguagem de marcação e é o bloco de construção mais básico do WEB. O documento é baseado em tags de texto que dividem os texto e dão funções e características aos texto da página. Abaixo iremos falar sobre a estrutura básica da página.


  1. DOCTYPE - indica para o navegador que o documento recebido está escrito em HTML.
  2. html - delimita o documento.
  3. head - delimita o cabeçalho. O conteúdo não é visível no navegador, mas contém instruções de comportamentos.
  4. meta - indica o conjuto de caracteres usado na página.
  5. title - título que aparecerá na aba do navegador.
  6. body -espaço com o conteúdo visível da página.

Diferenças entre listas ordenadas e não ordenadas

  1. A lista ordenada são ordenadas por números ou letras
  2. As listas ordenadas podem ter atributos como o type e start
